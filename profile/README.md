# KanTV AI

KanTV AI is an nonprofit software technical team which focus on learning&practising device-AI tech on <b>Android smartphone</b>. kantv-ai is founded on 02/12/2025.

# Roadmap

 - we currrently focus on <a href="https://github.com/zhouwg/kantv/blob/master/README-qnn.md">ggml-qnn</a>'s implementation which <a href="https://github.com/zhouwg/kantv/issues/121">launched on 03/29/2024</a> and <a href="https://github.com/zhouwg/kantv/issues/246">re-launched on 02/2025</a> on Android smartphone equipped with Qualcomm Snapdragon processors. we'd like to provide ggml-qnn's open source implementation and build public good for the great <a href="https://github.com/ggerganov/llama.cpp/">llama.cpp community</a>.
 - we'll take some efforts on <a href="https://github.com/pytorch/executorch">Meta's executorch</a> to help programmers and AI researchers deploy pytorch solution on Android smartphone more easily.
 - we'd like to provide other ggml backends for Android devices with mainstream mobile SoCs(such as UniSoC, MediaTek,...) if we can get necessary technical documents.
